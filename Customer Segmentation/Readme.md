Customer Segmentation Analysis

Objective:
To segment customers based on income, age, and spending behavior using machine learning.

Tools Used:
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, SQL

Steps Performed:

Data cleaning and preprocessing
Exploratory Data Analysis (EDA)
Feature scaling
K-Means clustering
Elbow method for optimal clusters

Results:

Identified 5 customer segments
Derived actionable insights for targeted marketing

Business Impact:

Helps improve marketing efficiency
Enables personalized campaigns
Increases customer retention
